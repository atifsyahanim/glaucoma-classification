# Glaucoma Classification Using Fundus Images

This project focuses on **glaucoma classification** using **retinal fundus images**.  
The goal is to assist ophthalmologists by providing a deep learning-based tool that can classify healthy vs glaucomatous eyes accurately and efficiently.

---

## Notebook

The main notebook contains:
- Data preprocessing and augmentation
- Training and fine-tuning of 4 transfer learning CNN (ResNet18) models
- Evaluation and performance analysis
- Grad-CAM for interpretability

---

## Dataset

Dataset used is from https://physionet.org/content/hillel-yaffe-glaucoma-dataset/1.0.0/Images/#files-panel
